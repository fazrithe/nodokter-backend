Cara install aplikasi upskills:

1. Extract file dalam folder htdocs
2. Buka kedalam terminal
3. Masuk dalam aplikasi cd/upskills
4. Composer install
5. npm run dev
6. php artisan key:generate
7. Buka file .env dan sesuikan database
9. php artisan migrate
10. php artisan db:seed
11. atau kalian dapat langsung import database menggunakan file upskill.sql
